HOW TO RUN THE SCRIPTS for XL-MS Protein assembly


#set up a modified version of CNS

1/ Download the sources of CNS 1.21 (http://cns-online.org/v1.2/)
2/ Unzip "CNS_modified_sources.tar" archive and copy the content in the "Sources" directory of CNS
3/ Compile CNS 

#set up the working directory

1/ Unzip the "Cross-links modeling.zip" archive
2/ add a symbolic link or an executable of CNS (named 'cns') in the unzipped directory
